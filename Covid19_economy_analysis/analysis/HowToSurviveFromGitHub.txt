[Before you work on your branch]

1. Make sure you are in the 'main' branch.
2. git pull
3. Switch it to your branch
4. git switch 'your branch name'
5. Check if you are in your branch
6. git pull (It will show everything is up to date.)
7. git merge origin/main

[After you are done in your end]

1. Check if you are in your branch
2. git add -A or git add -u
3. git commit -m "your message"
4. git push
5. Switch it to the main branch
6. git merge origin/'your branch name'
7. git push

[Always!!! Check GitHub if it is updated correctly]
